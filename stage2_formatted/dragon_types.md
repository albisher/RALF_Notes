# dragon_types

## Summary

```
Lists elemental, thematic, and age-based dragon types commonly used in fantasy lore.
```

## Details

> This file enumerates various dragon classifications, including elemental types (Fire, Ice, Lightning, etc.), thematic categories (Shadow, Poison), and age-based categories (Young, Wyrmling, Ancient). These types are often used in fantasy storytelling, role-playing games, or dragon lore systems to define traits, abilities, or roles of dragons.

## Usage

Referenced in lore, game mechanics, or character design to categorize dragons by type, origin, or stage of life. Example: A *Lightning Dragon* might have electrical-based attacks, while a *Wyrmling* is a young, immature dragon.
