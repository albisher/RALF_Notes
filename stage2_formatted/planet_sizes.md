# planet_sizes

## Summary

```
Describes a taxonomy of planetary size and type classifications used in astronomy and exoplanet studies.
```

## Details

> This file outlines a standardized naming convention for categorizing planetary sizes and types, ranging from sub-stellar objects like **Brown Dwarfs** to gas giants like **Super-Jupiters**. The taxonomy includes categories for terrestrial planets, ice giants, and substellar objects, reflecting their mass, composition, and orbital characteristics. It serves as a reference for comparing exoplanet sizes to known solar system bodies.

## Key Functions

### `Size Classification`

Assigns descriptive labels (e.g., *Earth-Size*, *Neptune-Size*) to planetary masses.

### `Composition Mapping`

Links terms like *Dwarf*, *Compact*, and *Fragment* to inferred physical properties (e.g., rocky vs. gaseous).

### `Comparative Analysis`

Enables direct comparisons between hypothetical exoplanets and solar system analogs.

## Usage

This taxonomy is used in:
- **Exoplanet research** to categorize discovered worlds.
- **Astrophysical modeling** for planet formation simulations.
- **Public outreach** (e.g., naming exoplanets) to convey scale intuitively.
