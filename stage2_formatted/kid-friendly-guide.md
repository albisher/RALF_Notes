# kid-friendly-guide

## Summary

```
A kid-friendly interactive web tool that simplifies story creation with visual prompts, emojis, and animations for young writers.
```

## Details

> Space Pearl is a **zero-pressure, visually engaging** platform designed for children to generate imaginative stories through a **four-step guided process**. It eliminates typing requirements by relying on **click-based selections** (e.g., story types, hero traits, settings) and includes **automatic saving** and sharing features. The system leverages **deterministic logic** to ensure consistent story outcomes, making it ideal for series-building. Emojis, animations, and playful design elements enhance accessibility and fun, while simplicity avoids overwhelming complexity.

## Key Functions

### `Story Type Selector`

Adventure, Fantasy, Space, Mystery, Friendship, Hero.

### `Hero Builder`

Name input + trait selection (Brave, Smart, Kind, etc.).

### `Setting Chooser`

Magical Castle, Space Station, Enchanted Forest, etc.

### `Automatic Story Generation`

Combines selections into a visual narrative.

### `Save/Share Functionality`

Stores stories locally and enables sharing via URL.

### `Deterministic Magic`

Ensures identical inputs produce identical stories (for series consistency).

## Usage

1. Access via `http://localhost:5173`.
2. Click **"Start Creating My Story!"** to begin.
3. Follow the **4-step workflow** (Type → Hero → Setting → Generate).
4. **Save/Share** the story or create another.
