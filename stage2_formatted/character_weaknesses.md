# character_weaknesses

## Summary

```
List of common character flaws affecting emotional, psychological, and behavioral traits.
```

## Details

> This file enumerates intrinsic weaknesses that may define a character’s struggles, including cognitive, emotional, and social challenges. Weaknesses like self-doubt and overthinking reflect internal conflicts, while physical limitations and social awkwardness highlight external barriers. These traits often drive narrative tension, shaping how characters interact with others and face challenges.

## Usage

This list can be used for:
- Character development (e.g., defining backstory, motivations, or arcs).
- Worldbuilding (e.g., societal impacts of these traits).
- Story plotting (e.g., conflicts arising from weaknesses).
