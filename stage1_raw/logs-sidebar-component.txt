### SUMMARY
Component for displaying and filtering communication logs in a right sidebar with embedded mini-map functionality.

### DETAILS
`LogsSidebarComponent` is a Vue.js-based UI component that renders a right sidebar displaying communication logs when `currentView` is set to `'list'`. The component integrates a mini-map (using OSM imagery) and a GPS preset selector. The map section includes a crosshair overlay for GPS positioning, while the dropdown allows users to select predefined GPS locations. The sidebar is structured with a header, map container, and content area, with conditional rendering of the map and GPS controls.

### KEY_FUNCTIONS
- **`onGpsPresetChange`**: Handles selection changes in the GPS preset dropdown.
- **`selectedGpsPreset`**: Stores the currently selected GPS preset (likely a reactive data property in Vue).
- **Mini-map rendering**: Dynamically displays an OSM map image with a crosshair overlay for GPS centering.
- **Conditional rendering**: Shows/hides the sidebar and map based on `currentView` state.

### DEPENDENCIES
Vue.js (for reactivity and templating), Cesium/OSM imagery (for map rendering), potentially external libraries for map styling or event handling.

### USAGE
To use this component:
1. Set `currentView` to `'list'` to render the sidebar.
