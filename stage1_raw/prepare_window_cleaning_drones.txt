### SUMMARY
Generates drone configurations for window cleaning missions, defining large and small scout drones with detailed specs.

### DETAILS
This script initializes a `WindowCleaningDronePreparer` class to programmatically create and save reusable configurations for window-cleaning drones. It defines two drone types:
- **Large scout drones** (full-featured with all sensors/addons) capable of autonomous mission completion.
- **Small scout drones** (minimal sensors) for distributed work in teams.

The class handles directory setup and generates a JSON configuration dictionary for each drone type, including specs like mass, battery life, sensors, and capabilities. The `create_large_scout_config` method populates a comprehensive configuration for autonomous missions, while the class structure supports extensibility for additional drone types.
